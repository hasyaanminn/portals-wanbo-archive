# portals-wanbo-archive

## 项目简介

portals-wanbo-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库旨在整理和存储一些零散的网页资源，不针对任何特定域名或具体网站。

## 目录说明

```
.
├── index.html          # 仓库首页/导航页
├── pages/              # 存放归档的 HTML 页面
│   ├── page1.html
│   └── page2.html
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 所有归档页面均为独立 HTML 文件，放置在 `pages/` 目录下。
- 每个页面可包含独立样式和脚本，不依赖外部网络资源（除非必要）。
- 页面内容可能来自不同来源，经过整理后统一归档。
- 归档不涉及任何商业用途，仅作为资料保存。
